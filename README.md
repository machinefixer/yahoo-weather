# yahoo-weather

A clone of Yahoo Weather iOS APP

### Requirements

* Xcode8+

### Quick Start

* Clone this repo
* `cd yahoo-weather`
* `Pod install` 
* Open YahooWeather.xcworkspace
* Add your [OpenWeatherMap](http://openweathermap.org) API Key to the file `api_key`, place your key file in project root directory.
* Build and Run
